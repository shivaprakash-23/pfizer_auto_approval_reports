PR Validation — False Positive Analysis

Prepared June 3, 2026  ·  Based on 145 PRs evaluated  ·  Prompt changes deployed June 2, 2026
▲

The agentic validation workflow currently flags 34 of 145 PRs as false positives — a 23% false positive rate. Two prompt changes shipped on June 2 targeting the top two error sources (Steps 7 & 11) are projected to reduce this to 9% (14 FPs). Actual outcomes will be confirmed with June 5 data.

📄
Data scope — PR types included

This report covers false positive analysis across the following three PR categories only:

Standard PRs Country — NAM-US High Match PRs

PRs outside these categories are excluded from all metrics and error counts in this report.
